# Diamond Warriors Baseball League Management Platform

## Description
**Diamond Warriors Baseball League** is a comprehensive web application designed to promote a baseball league and streamline league management. Built with a modern technology stack, it ensures a seamless and responsive user experience across devices.

The frontend leverages React with Vite, HTML, Tailwind CSS, and JavaScript, creating an engaging interface for users. The backend is powered by Firebase, providing robust data management and secure authentication using OAuth 2.0 for Google Sign-In. This setup offers a reliable and scalable environment for league operations.

The platform allows parents to sign up, view team pages, practice calendars, and volunteer opportunities. An admin view is available for league organizers to manage team assignments and keep information up-to-date effortlessly.

## Installation Instructions
**Required Programs:**
- Node.js
- Firebase CLI

**Steps to get the project running:**
1.	Clone the repository.
2.	Install dependencies with npm install.
3.	Run npm run dev to start the development server.
4.	Open the application at http://localhost:3000/.
5.	Done!

## Technologies Used
- React with Vite
- HTML
- JavaScript
- Tailwind
- Firebase
- O-Auth 2.0 for Google Sign-In

## Usage
The Baseball League Management Platform offers a user-friendly experience for parents and administrators. It works seamlessly on desktops, phones, and tablets. The Admin View allows for easy management of team assignments, practice schedules, and volunteer sign-ups.
